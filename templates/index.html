$body$ LiquidHaskell (LH) is a static verifier for Haskell that enriches Haskell's types with refinement predicates that let you enforce critical correctness properties at compile time.

Guarantee Functions are Total


LH warns you that head is not total as it is missing the case for [] and checks that it is total on NonEmpty lists.

The totality requirement is propagated to uses of head. These can be verified by ensuring the arguments are NonEmpty.

Keep Indexes and Pointers Within Bounds



LH lets you avoid off-by-one errors that can lead to crashes or buffer overflows.

Dependent refinements let you write contracts on multiple inputs, e.g. that dotProduct is only defined for equal-sized vectors.
